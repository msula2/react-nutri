Class: App

App()

Represents the state of the App component. The App component uses a Hash router to ensure that client side routing takes place. Adding a hash causes the server to ignore parts of the request, hence paths which are not mapped in the backend are taken care of using the HashRouter. For some paths such as the login, docs and register the naigation bar is not shown as well.
